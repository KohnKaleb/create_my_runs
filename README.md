## Run Finder, aka crafter.run

### Premise
The aim of this project is pretty simple:  <br />
(*) login to strava  <br />
(*) save your runs to our database  <br />
(*) look for community runs by elevation, distance, or whatever you care about  <br />
(*) discover new routes <br />
